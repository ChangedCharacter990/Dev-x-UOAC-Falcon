## Structure
- `extension/` — Chrome extension: Net Worth Predictor (docks your future net worth every time you scroll a short)
- `frontend/` — client app (web/mobile UI)
- `backend/` — API / server code
- `docs/` — notes, ideas, architecture, pitch deck source
- `scripts/` — setup or dev helper scripts
- `assets/` — images, design files, etc.

## Getting Started
1. Clone the repo
2. Fill in `frontend/` and `backend/` once the stack is chosen
3. Copy `.env.example` to `.env` and fill in secrets (never commit `.env`)

## Running the Chrome extension
1. Open Chrome and go to `chrome://extensions`
2. Toggle **Developer mode** on (top-right)
3. Click **Load unpacked** and select the `extension/` folder
4. Pin the extension to your toolbar
5. Visit YouTube Shorts, TikTok, or Instagram Reels and scroll — click the toolbar icon to watch your net worth drop

See `extension/README.md` for details on how detection works and how to tune it.
